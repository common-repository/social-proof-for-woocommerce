== Description ==

Social Proof for WooCommerce is social proof plugin for WooCommerce websites.

It is an eCommerce tool that lets websites show their latest orders. The plugin is built for WooCommerce websites, and is compatible with any website running on WooCommerce.

**GENERIC FEATURES:**

* Easy to set up, simple settings
* A neat design of the popups
* No signups or 3rd party integration required
* Fits equally on all screen sizes
* A lightweight plugin that doesn't affect your website performance

**CORE FEATURES:**

* Choose the number of orders to display
* Set a minimum product price
* Choose whether to repeat the popup or stop after the set number of orders are displayed
* Choose whether to display the "Verified by Ilmigo" label or not
* Choose whether to display the order time (i.e. days ago, minutes ago, etc)?
* Configure delay, timing, duration, etc.
* Set expiration time for a cookie to prevent the popup from trigger after it is closed by the user
* Options to show the popups sitewide, on specific posts/pages, categories, etc.
* An option for resetting the settings to defaults
* A graph to see the click analytics visually
* Show real names and locations of buyers
* Popups link directly to product URLs

== Installation ==

* The plugin is an extension of WooCommerce plugin. So make sure your site has WooCommerce installed prior to installing this plugin.
* Install Social Proof for WooCommerce via the plugin directory, or upload the files manually to your server.
